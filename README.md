# Central School of Commerce Typing Portal

This is the project directory for the premium typing test platform designed for the **Central School of Commerce**.

## Quick Start Setup

1. **Install Node.js**: Install Node.js LTS from [nodejs.org](https://nodejs.org/).
2. **Install Git**: Install Git for Windows from [git-scm.com](https://git-scm.com/).
3. **Open Workspace**: Open this directory `C:\Users\LTI INSTITUTE\.gemini\antigravity\scratch\csc-typing-portal` in your IDE.
4. **Supabase Database**: Apply the SQL schema provided in `schema.sql` to your Supabase project.
5. **Run the App**:
   ```bash
   npm install
   npm run dev
   ```
